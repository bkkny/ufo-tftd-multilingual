# Multilingual patches for UFO: Enemy Unknown and X-Com: Terror From The Deep

## UFO: Enemy Unknown (aka. X-Com: UFO Defense) (DOS, 1993)

This patch adds support for Hungarian, Polish, Czech, Finnish, Danish and Spanish in addition to the languages available in the original release of the game.

Please note that you need to own the original game to be able to install the patch.

Extract the archive into your UFOEU directory and run UFOEU.BAT to select your language.

Download the patch [here](./ufoeu_6in1_language_patch.zip)

## X-Com: Terror From The Deep (DOS, 1995)

This patch adds support for Polish, Czech, Hungarian and Italian in addition to the languages available in the original release of the game. It also contains the latest official patch from the manufacturer (v2.1) that fixes a number of bugs in the game (readme included for details).

Please note that you need to own the original game to be able to install the patch.

Extract the archive into your TFTD directory and run TFTD.BAT to select your language.

Download the patch [here](./tftd_v21_lang.zip)
